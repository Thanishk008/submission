The data is automatically downloaded via kagglehub. No manual dataset download required.

The dataset is NOT stored in this directory. kagglehub caches it at:
    Windows:  C:\Users\<username>\.cache\kagglehub\datasets\mlg-ulb\creditcardfraud\
    Linux:    ~/.cache/kagglehub/datasets/mlg-ulb/creditcardfraud/

If you prefer a local copy, place creditcard.csv in this folder and it
will be loaded directly (see --data_path argument in each script).
